Sistema crítico de facturación electrónica que recibe y procesa más de 15,000 peticiones diarias de archivos de facturas. Implementado con Python para la lógica de negocio, MongoDB para almacenamiento de documentos, Oracle para integración con sistemas legados y Nginx como reverse proxy de alta disponibilidad.
